## Get Curve Spot Price


Get the spot price of tokens from by calling the [curve.fi](https://curve.fi/) swap router using foundry

## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test --match-path test/GetPrice.t.sol -vvv
```

### Format

```shell
$ forge fmt
```

### Gas Snapshots

```shell
$ forge snapshot
```
